# Unapi GSM Megafon
Модуль для работы с мобильным API оператора [Мегафон](https://api.megafon.ru/)

Являтся частью библиотеки [Unapi](https://github.com/xRubin/unapi)

```bash
$ composer require unapi/gsm-megafon
```
